## House Price Prediction App

A practical Machine Learning application that predicts property prices in real time based on key housing features. Built to simulate how real estate platforms estimate value—fast, clean, and reliable.

### The Problem

Real estate data is rarely clean or ready to use.

Missing values across important features
Inconsistent address formats
High-cardinality location data (hard to model directly)
Mixed property types and categories
Noisy and unstructured inputs affecting prediction accuracy

In short, the data was messy, unreliable, and not model-friendly.

### My Approach (The Fix)

I focused on turning raw data into something a machine learning model can actually trust.

Cleaned and handled missing values using Pandas & NumPy
Standardized and transformed categorical features
Applied Target Encoding for address-based insights
Used KMeans Clustering to group locations intelligently
Built and trained a regression model using Scikit-learn
Saved and deployed the model using Joblib
Developed an interactive UI with Streamlit

The goal wasn’t just accuracy—it was usability and speed in a real-world scenario.

### The Impact (The Result)
Improved prediction reliability by reducing data noise
Enabled instant price estimation for users
Simulated business value by helping identify underpriced and overpriced properties
Created a system that can support better investment decisions

In a real-world setting, this kind of model can help businesses save thousands by pricing correctly and spotting opportunities early.
